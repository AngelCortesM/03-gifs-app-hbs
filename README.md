# 🌟 GifsApp

GifsApp es una aplicación desarrollada con [Angular CLI](https://github.com/angular/angular-cli) (versión 19.2.8) que permite buscar y explorar GIFs animados de manera rápida y sencilla.

---

## 🚀 Características

- 🔍 **Búsqueda de GIFs**: Encuentra GIFs animados de forma instantánea.
- ⚡ **Interfaz Reactiva**: La aplicación se actualiza automáticamente al modificar los archivos fuente.
- 📦 **Optimización para Producción**: Construcción optimizada para un rendimiento superior.

---

## 🛠️ Comandos Principales

### ▶️ Servidor de Desarrollo

Inicia un servidor local para desarrollo con:

```bash
ng serve
```

Abre tu navegador en `http://localhost:4200/` para ver la aplicación. Los cambios en el código se reflejarán automáticamente.

---

### 🏗️ Generación de Código

Angular CLI incluye herramientas para generar componentes, directivas, tuberías y más. Por ejemplo:

```bash
ng generate component nombre-del-componente
```

Para ver todos los esquemas disponibles:

```bash
ng generate --help
```

---

### 🏗️ Construcción del Proyecto

Compila el proyecto y genera los artefactos en la carpeta `dist/`:

```bash
ng build
```

Por defecto, la construcción está optimizada para producción.

---

### ✅ Pruebas Unitarias

Ejecuta las pruebas unitarias con el framework [Karma](https://karma-runner.github.io):

```bash
ng test
```

---

### 🧪 Pruebas End-to-End (e2e)

Para pruebas de extremo a extremo, utiliza:

```bash
ng e2e
```

> Nota: Angular CLI no incluye un framework e2e por defecto. Puedes elegir el que mejor se adapte a tus necesidades.
